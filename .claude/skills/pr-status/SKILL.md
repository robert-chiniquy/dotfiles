# PR Status

Check status of your open pull requests: comments, reviews, CI failures, recent activity.

Quick commands for: listing open PRs, checking all PR activity, getting PR details with checks and reviews, checking CI status, getting recent comments and review comments.

Batch operations: PRs with failures, PRs awaiting response, PRs ready to merge. Common workflows: morning PR check, respond to review.
